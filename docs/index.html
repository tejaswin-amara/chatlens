✦ System Installation Guide

Deploy ChatLens in your own secure environment in under 5 minutes.

Run the entire FastAPI backend, Next.js frontend, PostgreSQL database, and Redis Celery broker in containerized mode:

# 1. Clone the repository and configure env
git clone https://github.com/tejaswin-amara/chatlens.git
cd chatlens
cp .env.example .env  # Add your GEMINI_API_KEY from Google AI Studio

# 2. Build and launch services
docker-compose up -d --build

# 3. Open UI
# Navigate to http://localhost:3000

Run standard services locally on your host environment (Requires PostgreSQL with `pgvector` and Redis running):

# 1. Setup Backend environment
cd chatlens-v2/backend
python -m venv venv
source venv/bin/activate  # venv\Scripts\activate on Windows
pip install -r requirements.txt
uvicorn main:app --reload --port 8000

# 2. Run Celery workers (in separate terminal)
celery -A celery_worker.celery_app worker --loglevel=info

# 3. Build & Run Frontend
cd ../frontend
npm install
npm run dev  # Serves the site on http://localhost:3000
