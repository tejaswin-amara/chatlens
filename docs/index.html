Product Bento Overview

ChatLens utilizes a highly decoupled, modern tech stack designed for security and speed.

🧠

LLM Analysis Engine

Integrates seamlessly with Gemini models using Google's generative-ai SDK. Handles massive chat transcripts easily by parsing context streams in configurable timeline chunks.

📁

Multi-Platform

Ingests standard JSON exports from Telegram Desktop or raw WhatsApp text file backups on the fly.

🔍

pgvector Indexes

Executes semantic lookups across chat messages in PostgreSQL utilizing the pgvector extension for high-performance retrieval.

🎨

Glassmorphic Web UI

React frontend built on Next.js App Router. Features clean, hardware-accelerated transitions, shimmering skeletons, empty-state prompts, and fully responsive layouts.

System Setup Guide

Configure and launch ChatLens in a few terminal steps.

Spin up PostgreSQL, Redis, backend Celery tasks, and Next.js frontend services concurrently:

# 1. Clone the repository and configure env
git clone https://github.com/tejaswin-amara/chatlens.git
cd chatlens
cp .env.example .env  # Add your GEMINI_API_KEY from Google AI Studio

# 2. Build and launch services
docker-compose up -d --build

# 3. Open UI
# Navigate to http://localhost:3000

Run FastAPI server and Next.js locally (requires local PostgreSQL + pgvector and Redis):

# 1. Start Python Backend server
cd chatlens-v2/backend
python -m venv venv
source venv/bin/activate  # venv\Scripts\activate on Windows
pip install -r requirements.txt
uvicorn main:app --reload --port 8000

# 2. Start Celery worker in separate terminal
celery -A celery_worker.celery_app worker --loglevel=info

# 3. Start Next.js Web UI
cd ../frontend
npm install
npm run dev
