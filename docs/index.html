ChatLens v2.0 🔍

Privacy-first, AI-powered Telegram & WhatsApp chat analyzer using Google Gemini, Next.js, and FastAPI.

✦ System Overview

ChatLens is an interactive workspace designed to help you index, search, and run advanced LLM analytics on your conversational archives. Under v2.0, the core is split into a premium glassmorphic React frontend and a Docker-orchestrated FastAPI/Celery parser and database server.

Next.js 16 FastAPI Google Gemini PostgreSQL + pgvector Celery Redis Telethon (Live Sync) Docker Compose
📁
Multi-Platform Ingest
Import standard chat export formats from Telegram (.json) and WhatsApp (.txt). All parsing happens asynchronously via Celery workers.
⚡
Live Account Sync
Connect directly to Telegram using your own App API credentials. Authorize securely and import direct message and group logs on demand.
🧠
Advanced AI Analysis
Leverage Gemini's massive context window to compile global summaries, identify toxic patterns, map relationships, and run vibe checks.
📊
Spotify-Wrapped Stats
Get quantitative activity indicators. Track engagement times, hourly activity heatmaps, and unlock fun custom badges (e.g., Monologuer, Speed Demon).

✦ Deploying Locally

Because ChatLens processes private conversations and uses persistent datastores, it must be deployed locally or in a private cloud environment:

# 1. Clone the repository
git clone https://github.com/tejaswin-amara/chatlens.git
cd chatlens

# 2. Configure environment
cp .env.example .env
# Open .env and add your GEMINI_API_KEY from Google AI Studio

# 3. Start with Docker Compose
docker-compose up -d --build

Once standard services spin up, navigate to http://localhost:3000 to launch the web dashboard interface.
