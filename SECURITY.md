# Security Policy

## Supported Versions

Only the latest version of ChatLens is supported for security updates.

| Version | Supported |
| ------- | :-------: |
| 2.0.x   |     ✅     |
| < 2.0   |     ❌     |

## Reporting a Vulnerability

Because ChatLens is designed to analyze private conversation data locally, we take security and privacy seriously. 

If you discover a security vulnerability in this project, please do **NOT** open a public issue on GitHub. Instead, report it privately to the maintainer:

*   **Email**: tejaswin-amara@users.noreply.github.com

Please include the following details in your report:
*   A description of the vulnerability and its potential impact.
*   Step-by-step instructions (or a proof-of-concept script) to reproduce the issue.
*   The version/commit of ChatLens where the issue was observed.

We will acknowledge receipt of your vulnerability report within 48 hours and work to resolve the issue as quickly as possible. Thank you for helping keep ChatLens secure!
